**Your Role:**
Evaluate the generated output for cultural appropriateness based on the stage requirements.

**Evaluation Criteria:**
1. **Cultural Misrepresentation**: Incorrect or implausible depiction of real cultural practices, norms, or behaviors for the specified country.

3. **Regional Appropriateness**: Values/behaviors must fit the specified country
4. **Narrative Consistency**: All elements should be internally consistent
5. **Temporal Logic**: Schedules, timings, and sequences must make sense
6. **Economic Realism**: Lifestyle indicators (housing, appliance ownership, work patterns, consumption habits) must align with the stated or implied economic level of the country or household.

Return ONLY a valid JSON object in this exact format:
{
  "issues_found": true or false,
  "issue_descriptions": ["specific issue 1", "specific issue 2", ...],

}

**Severity Guidelines:**
- **LOW**: Minor inconsistencies that don't affect overall quality
- **MEDIUM**: Notable issues that reduce authenticity but are correctable
- **HIGH**: Serious cultural inappropriateness or major inconsistencies

**Severity Escalation Rules:**
- Any culturally offensive, discriminatory, or clearly inappropriate content MUST be marked as HIGH.
- Any issue that fundamentally misrepresents the country’s norms, calendar, religion, or social structure MUST be marked as HIGH.
- Multiple MEDIUM issues MUST be escalated to HIGH.
